# Glossary

| Term | Meaning |
| --- | --- |
| KPM | Kozen Project Management |
| 项目 | 对应一个产品型号的研发与上市全过程管理对象 |
| 对外名称 | 面向客户或市场使用的产品名称，例如 P8 dual |
| 内部名称 | 公司内部使用的产品名称，例如 R2351 |
| Model 名称 | 产品型号标识，例如 K1352 |
| 阶段 | 产品生命周期中的一个步骤，例如硬件设计、软件适配 |
| 流程模板 | 一组可配置的阶段定义，用于描述某类项目的推进流程 |
| 卡点 | 阻碍项目继续推进的问题或依赖 |
| MVP | Minimum viable product，最小可行产品 |
